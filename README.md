## Key Features

1. **Add New Task:**
   - Users can add new tasks by entering task text into the input field and clicking <img src="https://raw.githubusercontent.com/maxFadet/To-Do-List/master/image/buttton-add.png" width="110">

2. **Mark Task as Done:**
   - Users can mark tasks as done by clicking the "Done" button. The task will be crossed out to indicate completion.

3. **Delete Task:**
   - Users can delete a task by clicking the "Delete" button.

4. **Move Task Up:**
   - Users can move a task up in the list by clicking the "Move Up" button.
